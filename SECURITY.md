# Security Policy

## Supported Versions

The current `main` branch is supported during early development.

## Reporting a Vulnerability

Please do not open a public issue for vulnerabilities that expose private data or secrets.

Use GitHub's private vulnerability reporting feature if it is enabled, or contact the maintainer privately through the address listed in the repository profile.

## Data Handling

`maintainer-signal` is designed to analyze local repository metadata. It should not collect API keys, credentials, private tokens, or personal identity documents.
