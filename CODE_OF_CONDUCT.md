# Code of Conduct

This project follows a simple standard: be respectful, constructive, and specific.

Maintainer tools are often used during stressful work. Contributions and discussions should help maintainers make better decisions without shaming people for missing documentation, tests, or process.

Unacceptable behavior includes harassment, personal attacks, hate speech, and attempts to publish private information.

If you need to report a concern, contact the maintainers through the private security or maintainer contact channel listed in the repository.
